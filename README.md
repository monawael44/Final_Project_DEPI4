# Final_Project_DEPI4

1.Project Name :
MTA Daily Ridership
  
2.Project Idea 
This project analyzes daily public transportation ridership data o explore ridership trends, seasonal patterns, and usage behavior over time. The project includes data cleaning, exploratory analysis, and time-series forecasting to predict ridership for the upcoming month. Insights are presented through visualizations and an interactive dashboard to support data-driven transportation planning.

4.Project Plan 
Week 1 – Data Preparation
Build a simple data model
Clean and preprocess daily ridership data
Handle missing values and date formatting
Week 2 – Exploratory Data Analysis
Analyze daily, weekly, and monthly ridership trends
Identify seasonality and peak usage periods
Compare ridership across different transit services
Week 3 – Forecasting
Apply time-series forecasting techniques
Predict total ridership for the next month
Visualize forecast results and trends
Week 4 – Visualization & Reporting
Build an interactive dashboard (Tableau)
Summarize insights and key findings
Prepare final project documentation for GitHub
